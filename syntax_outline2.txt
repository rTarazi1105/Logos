Syntax outline 2

Basic types:
- bool
- int
- statement
	- can be string
- value
	- definition: statement
- relation([value; n]) -> statement
	- inputs: [value; n]
	- e.g. "function" and "variable"
- operation([statement; n]) -> statement
- property<Type>([Type; n]) -> statement
	- type can be class or struct

Arrays: [Type; n]
